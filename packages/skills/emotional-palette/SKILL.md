# 情绪调色板 (Emotional Palette)

## Core Capabilities

1. **多维情绪检测** — 分析快乐、愤怒、焦虑、兴奋、平静、沮丧等 8+ 种情绪
2. **情绪时间线** — 构建按时间排列的情绪变化图谱
3. **抽象艺术可视化** — 将情绪数据转化为抽象画作（颜色=情绪，形状=强度）
4. **触发因素分析** — 识别导致情绪变化的触发因素（人物、话题、场景）
5. **情绪报告** — 生成周/月情绪健康报告，包含趋势和建议
6. **情绪预测** — 基于历史模式预测未来可能的情绪波动

## User Experience Flow

```
用户输入 → 模型理解 → 执行动作 → 输出结果

1. 用户触发"分析我的情绪变化"
2. 系统扫描指定范围的录音，执行情绪分析
3. LLM 对每段录音标注多维情绪标签
4. 构建情绪时间线数据
5. 生成抽象艺术可视化（前端渲染）
6. 输出情绪洞察报告
```

## Technical Implementation

| 组件 | 技术方案 | 说明 |
|------|----------|------|
| 情绪分析 | LLM 多标签分类 | 识别多种情绪维度 |
| 语音情绪 | 音频特征提取 | 语速、音量、语调 |
| 文本情绪 | NLP 情感分析 | 用词、句式、标点 |
| 可视化 | Canvas / SVG | 抽象艺术渲染 |
| 时间序列 | Pandas + 时间分析 | 情绪趋势追踪 |
| 报告生成 | LLM + 模板 | 情绪洞察报告 |

### 情绪维度模型

```yaml
emotional_dimensions:
  primary:
    - joy: 快乐 # 暖色调，上升曲线
    - anger: 愤怒 # 红色系，尖锐形状
    - anxiety: 焦虑 # 黄色系，波动曲线
    - excitement: 兴奋 # 橙色系，跳跃形状
    - calm: 平静 # 蓝色系，平滑曲线
    - sadness: 沮丧 # 灰色系，下沉曲线
    - surprise: 惊讶 # 紫色系，突变形状
    - confidence: 自信 # 金色系，稳定上升
  
  intensity: 0-10 # 情绪强度
  valence: -1 to 1 # 正负情绪
  arousal: 0-10 # 唤醒度（平静到激动）
```

### 可视化映射规则

```python
EMOTION_COLORS = {
    "joy": "#FFD700",       # 金色
    "anger": "#FF4444",     # 红色
    "anxiety": "#FFB347",   # 橙黄
    "excitement": "#FF6B35",# 橙色
    "calm": "#4ECDC4",      # 青色
    "sadness": "#95A5A6",   # 灰色
    "surprise": "#9B59B6",  # 紫色
    "confidence": "#F39C12" # 金色
}

def render_emotional_art(emotion_data):
    """将情绪数据渲染为抽象画"""
    canvas = []
    for point in emotion_data:
        color = EMOTION_COLORS[point.primary_emotion]
        size = map_intensity(point.intensity, 10, 100)
        shape = get_emotion_shape(point.emotion)
        canvas.append({
            "x": point.timestamp,
            "y": point.valence * canvas_height / 2,
            "color": color,
            "size": size,
            "shape": shape
        })
    return canvas
```

```

【本周情绪地图】

周一 ████████░░ 焦虑(7) → 自信(8) — 从担心项目进度到评审通过的转变
周二 ██████░░░░ 平静(6) — 常规工作日，情绪稳定
周三 █████████░ 兴奋(9) → 愤怒(4) — 新功能演示兴奋，但被质疑后情绪波动
周四 ██████░░░░ 快乐(6) — 团队聚餐，氛围轻松
周五 █████████░ 自信(9) — 周报总结，对本周成果满意

【抽象情绪画】
已生成本周情绪可视化图：
- 主色调：金色（快乐）+ 青色（平静），占比 65%
- 波动点：周三下午（红色突变 = 愤怒触发）

【洞察】

```
