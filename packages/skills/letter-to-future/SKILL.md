# 给未来的信 (Letter to Future)

## Core Capabilities

1. **语音/文字录制** — 支持录音或文字形式写给未来的自己
2. **时间胶囊封装** — 将内容加密存储，设定投递时间
3. **状态快照** — 记录当前的生活状态、工作、情绪、目标
4. **定时投递** — 按设定时间触发投递通知
5. **变化对比** — 投递时自动对比当前状态与过去快照
6. **成长报告** — 生成"你变了/没变"的对比分析报告

## User Experience Flow

```
用户输入 → 模型理解 → 执行动作 → 输出结果

1. 用户触发"给未来的自己写封信"
2. 系统引导用户录制内容（语音/文字）
3. LLM 辅助生成当前状态快照（工作、情绪、目标）
4. 封装为时间胶囊，设定投递时间
5. 到期后推送通知，附带变化对比分析
```

## Technical Implementation

| 组件 | 技术方案 | 说明 |
|------|----------|------|
| 内容录制 | 录音 / 文字输入 | 用户输入方式 |
| 状态快照 | LLM + 多源数据 | 聚合当前状态数据 |
| 加密存储 | AES-256 加密 | 保护隐私内容 |
| 定时任务 | 调度系统 | 到期触发投递 |
| 变化对比 | LLM 对比分析 | 现状 vs 过去 |
| 报告生成 | 模板 + LLM | 成长对比报告 |
| 通知推送 | 飞书/手机推送 | 投递通知 |

## Usage Examples

### 示例 1：一年期时间胶囊

```
用户：给一年后的自己写封信
系统：好的，请先录一段话给未来的自己。（支持语音或文字）

用户：（录音）"嘿，一年后的我，希望你还在做AI硬件这个方向。
我现在刚启动AI录音卡项目，压力挺大但很兴奋。
希望一年后产品已经上市了，用户反馈不错。
还有，记得多运动，别老加班..."

系统：已录制完成。让我帮你生成当前状态快照...

【时间胶囊已封装】

封存时间：2026-05-23
投递时间：2027-05-23
投递方式：飞书消息 + 邮件

当前状态快照：
- 工作：小米创新产品部，AI录音卡项目刚启动
- 情绪：兴奋(60%) + 焦虑(30%) + 平静(10%)
- 目标：产品成功上市、用户反馈好、多运动
- 个人关键词：新挑战、学习中、充满期待

已加密存储，一年后将准时投递。
```

### 示例 2：到期投递

```
系统：（一年后投递通知）

📬 时间胶囊投递通知

一年前的你录了一段话，现在到了打开的时候。

【一年前的你】
"嘿，一年后的我，希望你还在做AI硬件这个方向..."
（完整录音播放）

【一年后的对比】

变化的：
- 你确实还在AI硬件方向，而且已经从产品经理成长为产品线负责人
- 产品上市了，首年销量 12 万台，用户好评率 87%
- 新增了AI陪伴玩具产品线（一年前没预料到）

没变的：
- 你依然热爱这个领域，热情不减
- 你还是习惯性加班（这一点真的没变...）
- 你的沟通风格保持一致——数据驱动、逻辑清晰

没达成的：
- "多运动"——运动频率从每周 2 次降到了每月 1 次
- 建议：把运动加入日历，像会议一样对待

【整体评价】
过去一年，你在事业上取得了显著进步，产品从0到1成功落地。
在个人健康方面还有提升空间。
继续保持对工作的热情，同时关注身体。
未来的你，会感谢现在努力的自己。
```
