# 简历生成 (Resume Generator)

## Core Capabilities

1. **工作内容提取** — 从录音中识别角色职责、项目经历、日常任务，构建工作画像
2. **技能图谱构建** — 自动归纳硬技能（技术、工具）和软技能（沟通、管理、决策）
3. **成就量化** — 从对话中提取可量化的成果（数字、百分比、增长率、节省金额）
4. **STAR 模型构建** — 将经历组织为 Situation-Task-Action-Result 结构
5. **多版本简历生成** — 根据目标岗位自动调整简历侧重点（技术岗 / 管理岗 / 创业方向）
6. **关键词优化** — 基于 ATS（自动筛选系统）优化简历关键词密度

## User Experience Flow

```
用户输入 → 模型理解 → 执行动作 → 输出结果

1. 用户触发"帮我写简历"，可选：指定目标岗位或让AI推荐
2. 系统扫描全部录音，按时间线和项目维度提取工作内容
3. LLM 分析提取：角色、技能、项目、成果、关键词
4. 构建简历骨架（个人信息 + 工作经历 + 技能标签 + 项目亮点）
5. 根据目标岗位调整侧重点和措辞
6. 输出简历草稿（Markdown / 飞书文档 / PDF）
```

## Technical Implementation

| 组件 | 技术方案 | 说明 |
|------|----------|------|
| 全文检索 | 关键词 + 语义搜索 | 定位相关录音片段 |
| 内容提取 | LLM 信息抽取 | 识别技能、角色、成果 |
| 成就量化 | NER + 数字识别 | 提取可量化指标 |
| 简历生成 | 模板引擎 + LLM | 结构化输出 |
| ATS 优化 | 关键词匹配算法 | 目标岗位关键词覆盖 |
| 输出格式 | Markdown / DOCX / PDF | 多格式导出 |
