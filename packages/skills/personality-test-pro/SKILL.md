# 性格测试Pro (Personality Test Pro)

## Core Capabilities

1. **语言模式分析** — 分析用词习惯、句式结构、表达风格，推断思维模式
2. **决策风格画像** — 从决策场景中提取决策模式（分析型/直觉型/协调型/指令型）
3. **情绪反应图谱** — 分析不同场景下的情绪反应模式和触发因素
4. **多框架评估** — 结合 MBTI、DISC、大五人格等多个框架交叉分析
5. **社交风格分析** — 从对话互动中提取社交倾向（内向/外向、感性/理性）
6. **成长型洞察** — 识别性格中的成长空间和潜在发展方向

## User Experience Flow

```
用户输入 → 模型理解 → 执行动作 → 输出结果

1. 用户触发"分析我的性格"
2. 系统扫描全量录音（或指定范围）
3. LLM 从多个维度分析行为模式：
   - 语言风格 → 思维模式
   - 决策场景 → 决策风格
   - 互动模式 → 社交风格
   - 情绪反应 → 情感模式
4. 生成多框架性格报告
5. 提供性格发展建议
```

## Technical Implementation

| 组件 | 技术方案 | 说明 |
|------|----------|------|
| 语言分析 | LLM 文本分析 | 用词、句式、表达风格 |
| 决策分析 | LLM 场景提取 | 从决策场景中提取模式 |
| 情绪分析 | LLM + 音频 | 情绪反应模式识别 |
| 多框架评估 | LLM 知识库 | MBTI / DISC / 大五人格 |
| 报告生成 | 模板 + LLM | 结构化性格报告 |
| 可视化 | 雷达图 / 饼图 | 性格维度可视化 |

```
- 可控风险下愿意尝试（如新渠道测试）
- 不可逆决策极度谨慎（如大规模投资）

【建议】
- 你的决策风格适合复杂项目管理
- 可以在直觉决策后增加"假设检验"环节，降低盲点
```

### 示例 3：社交风格分析

```

系统：分析 42 条包含社交互动的录音...

【社交风格画像】

社交能量：
- 你属于"充电型社交者"——需要独处时间恢复社交能量
- 最佳社交时长：2-3小时（之后效率显著下降）
- 最佳社交形式：1:1 深度对话 > 小组讨论 > 大型社交

沟通风格：
- 深度优先：更愿意聊有深度的话题，而非寒暄
- 主动提问：你更擅长通过提问引导对话
- 倾听者倾向：说话时间占 35%（低于平均 50%）

冲突处理：
- 回避型 30% + 协作型 60% + 竞争型 10%
- 你倾向于先理解对方再表达自己
- 但在原则问题上会坚定立场

【成长建议】
- 适当增加"闲聊能力"，这在商务社交中很重要
- 练习在大型场合中主动发言（从提一个问题开始）
```
