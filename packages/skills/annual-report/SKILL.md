# 年度述职自动生成 (Annual Report)

## Core Capabilities

1. **全年录音聚合** — 按时间线汇总所有录音记录，自动分类（项目会议、1:1沟通、头脑风暴、汇报等）
2. **成果提取引擎** — 从对话中识别项目里程碑、关键决策、量化成果（数字、百分比、时间节点）
3. **成长轨迹建模** — 对比年初与年末的能力表现，生成成长曲线和能力雷达图
4. **述职报告生成** — 输出符合企业述职格式的结构化报告（PPT大纲 / 飞书文档）
5. **金句收集** — 提取录音中个人表达的精彩观点、方法论、金句作为报告亮点
6. **数据可视化** — 自动生成工作量分布、项目占比、时间投入等图表

## User Experience Flow

```
用户输入 → 模型理解 → 执行动作 → 输出结果

1. 用户触发"生成年度述职"，选择时间范围（默认：最近12个月）
2. 系统查询录音数据库，按月份/项目/类型聚合所有录音
3. LLM分批处理录音转写内容，提取：
   - 项目名称、参与角色、关键成果
   - 决策节点与影响
   - 量化指标（数字、进度、增长率）
   - 个人成长信号（新技能、方法论、领导力表现）
4. 生成述职报告草稿（含大纲、正文、数据支撑）
5. 用户确认/微调后，输出最终报告（飞书文档 / PDF）
```

## Technical Implementation

| 组件 | 技术方案 | 说明 |
|------|----------|------|
| 录音检索 | 时间范围 + 标签过滤 | 聚合全年录音 |
| 内容提取 | LLM 信息抽取 | 从转写文本中提取结构化数据 |
| 成果量化 | NER + 规则引擎 | 识别数字、指标、百分比 |
| 报告生成 | 模板引擎 + LLM | 企业述职模板 + AI 生成内容 |
| 可视化 | ECharts / D3.js | 工作量分布图、成长雷达图 |
| 输出格式 | 飞书文档 / PPT / PDF | 多格式导出 |

- **情绪调色板** — 补充工作情绪数据，展示压力分布
- **职业教练** — 将述职数据输入职业发展分析
- **谈判复盘器** — 提取谈判成果作为年度成就素材
