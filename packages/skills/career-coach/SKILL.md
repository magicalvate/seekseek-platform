# 职业教练 (Career Coach)

## Core Capabilities

1. **沟通风格分析** — 分析录音中的表达模式：语速、逻辑性、说服力、情绪控制
2. **领导力评估** — 基于会议录音评估决策风格、团队影响力、冲突处理能力
3. **职业路径规划** — 结合当前能力和目标，生成阶段性职业发展路线图
4. **短板识别** — 找出沟通、决策、协调等方面的薄弱环节
5. **话术训练** — 针对具体场景（汇报、谈判、演讲）提供话术模板和练习建议
6. **成长追踪** — 对比不同时间段的表现，量化成长进度

## User Experience Flow

```
用户输入 → 模型理解 → 执行动作 → 输出结果

1. 用户触发"给我职业发展建议"或"分析我的沟通风格"
2. 系统扫描指定范围的录音，提取沟通行为数据
3. LLM 用教练框架（GROW模型、非暴力沟通等）分析表现
4. 生成教练报告：优势识别 + 短板分析 + 行动计划
5. 提供可执行的练习任务和反馈机制
```

## Technical Implementation

| 组件 | 技术方案 | 说明 |
|------|----------|------|
| 行为分析 | LLM 多维度分析 | 沟通模式、决策风格、情绪管理 |
| 评估框架 | GROW / 非暴力沟通 / DISC | 结构化教练方法论 |
| 对比分析 | 时间序列对比 | 成长追踪 |
| 报告生成 | 模板 + LLM | 教练报告输出 |
| 练习系统 | LLM 互动 | 话术练习和反馈 |
| 语音分析 | 语音特征提取 | 语速、停顿、语调分析 |

## Usage Examples

### 示例 1：沟通能力分析

```
用户：分析一下我最近的沟通风格
系统：分析最近 30 条录音的沟通行为...

【沟通风格画像】

优势：
1. 逻辑性强 — 在方案讨论中，你的表达结构清晰，论据充分
2. 善于倾听 — 在 80% 的对话中，你会先复述对方观点再回应
3. 数据驱动 — 习惯用数据支撑观点（"这个方案能降低30%成本"）

待提升：
1. 情绪管理 — 在第14分钟的讨论中，当方案被质疑时语速加快35%，
   建议使用"暂停-呼吸-重构"技巧
2. 总结收尾 — 60%的会议缺少明确的结论和下一步行动，
   建议每次会议结尾使用"三句话总结法"

【本周练习】
练习1：每次会议结束前，用"总结+下一步+时间点"格式收尾
练习2：当感到被挑战时，先说"让我想一下"，再回应
```

### 示例 2：领导力评估

```
用户：我想知道自己适不适合转管理，帮我评估领导力
系统：基于你的录音进行领导力评估...

【领导力评估报告】

GROW 模型分析：

Goal（目标）— 你展现出清晰的方向感
- 在团队会议中，你能为讨论设定清晰框架（得分 8.2/10）
- 但在资源有限时的优先级排序还需加强（得分 6.5/10）

Reality（现状）— 当前领导力表现
- 决策风格：分析型（偏好数据和论证），适合复杂项目
- 影响力来源：专业能力 > 职位权力（健康的影响模式）
- 冲突处理：回避倾向 40%、协作倾向 45%、竞争倾向 15%

Options（选择）— 建议发展方向
1. 短期：提升"授权能力"（当前过于亲力亲为）
2. 中期：培养"战略思维"（从执行者向规划者转变）
3. 长期：建立"教练式领导"风格

Action（行动）— 下一步
本周任务：在下次团队会议中，主动让一位同事主导某个议题
```

### 示例 3：演讲能力提升

```
用户：下周要做产品发布会演讲，帮我练习
系统：基于你过往的演讲/汇报录音，提供针对性建议...

【演讲能力分析】

历史表现：
- 最近一次产品汇报：15分钟，逻辑评分 8.5/10，感染力 6.2/10
- 主要问题：前5分钟节奏太快，观众还在"热身"你已进入核心内容

【演讲结构建议】
1. 开场（2分钟）— 用一个用户故事或痛点场景引入
2. 问题（3分钟）— 市场机会和用户需求
3. 方案（5分钟）— 产品介绍，节奏放慢
4. 亮点（3分钟）— 3个核心卖点，每个1分钟
5. 收尾（2分钟）— 愿景 + 行动号召

【节奏控制】
- 关键数据出现后，停顿 2 秒
- 每3分钟插入一个互动问题
- 避免连续使用超过3个技术术语
```
