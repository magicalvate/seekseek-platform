# 社交分析图谱 (Social Graph)

## Core Capabilities

1. **人物实体提取** — 从录音中识别所有提及的人名、角色、组织
2. **关系网络构建** — 构建 人-人、人-项目、人-概念 三维度关系图
3. **互动频率分析** — 统计与每个人的合作频率、沟通时长、最近互动时间
4. **关系动态追踪** — 追踪关系强度随时间的变化趋势（增强/减弱/断裂）
5. **影响力评估** — 分析每个人在对话中的影响力权重（发言占比、决策影响力）
6. **互动策略建议** — 基于关系数据提供维护关系的行动建议

## User Experience Flow

```
用户输入 → 模型理解 → 执行动作 → 输出结果

1. 用户触发"分析我的人脉关系"
2. 系统扫描全部录音，执行人名实体识别（NER）
3. LLM 分析关系类型（同事、上级、客户、供应商、朋友）
4. 构建关系图谱数据结构（节点 + 边 + 权重）
5. 生成可视化图谱（力导向图 / 关系矩阵）
6. 输出互动分析报告 + 关系维护建议
```

## Technical Implementation

| 组件 | 技术方案 | 说明 |
|------|----------|------|
| 人名识别 | NER + 上下文消歧 | 从录音中识别人名 |
| 关系分类 | LLM 分类器 | 判断关系类型和强度 |
| 图谱构建 | Graph Data Structure | 节点 + 边 + 属性 |
| 可视化 | D3.js 力导向图 / ECharts 关系图 | 交互式图谱展示 |
| 时间分析 | 时间序列分析 | 关系动态追踪 |
| 策略生成 | LLM 推理 | 互动策略建议 |

## Dependencies

| 依赖项 | 类型 | 必需 | 说明 |
|--------|------|------|------|
| 录音数据库 | 内部 | 是 | 全量录音转写内容 |
| LLM 推理服务 | 云端/本地 | 是 | 实体提取与关系分析 |
| 图数据库 | 内部 | 否 | 关系图谱持久化存储 |
| 可视化库 | 前端 | 是 | 交互式图谱展示 |
| 飞书 API | 外部 | 否 | 人员信息补充 |

## Integration Points

- **录音卡录音系统** — 获取录音内容用于人物识别
- **ASR 转写服务** — 说话人分离辅助人名识别
- **飞书通讯录** — 补充人员组织信息（部门、职位、汇报关系）
- **飞书日历** — 关联会议日程，补充互动场景
- **谈判复盘器** — 将谈判对手信息加入图谱节点
- **年度述职生成器** — 协作网络数据作为团队协作能力佐证
- **开场白建议** — 基于关系数据提供破冰话题
- **职业教练** — 分析社交网络对职业发展的影响
- **情绪调色板** — 关联不同人物互动中的情绪数据
